## Command For Termux
```bash
termux-setup-storage
apt update && apt upgrade
pkg install nodejs && pkg install ffmpeg && pkg install git && pkg install libwep && pkg install imagemagick
git clone https://github.com/RaditHyzer/XyzMD
cd XyzMD
pkg install yarn
yarn
npm start
```
